MeshLink

A polished Meshtastic desktop client for Windows. Monitor your mesh network, send messages, view telemetry, and manage nodes — all from one app.

Download for Windows
v1.0.0-beta.1 · Windows x64 · ~98 MB Beta

Mesh Chat

Send and receive messages across your Meshtastic mesh network with per-channel views.

Node Explorer

See every node on the mesh with battery, SNR, position, hop count, and online status.

Live Telemetry

Real-time battery, voltage, channel utilization, temperature, and environmental data.

Serial & TCP

Connect via USB serial or network TCP. Auto-scan, auto-reconnect, and baud-rate detection.

System Requirements

  • Windows 10 or later (x64)
  • A Meshtastic-compatible device (Heltec V3, T-Beam, RAK, etc.)
  • USB cable or network connection to your Meshtastic node
  • CP210x or CH9102 USB driver (usually auto-installed by Windows)
